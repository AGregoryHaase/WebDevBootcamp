Variables

VARIABLES ARE LIKE LABELS FOR VALUES

  • We can store a value and give it a name sot hat we can:
    • Refer back to it later
    • Use that Vlaue to do...stuff
    • Or change it later on

WE may be running some complicated tasks where we need to refer back to some former calculation.

To declare a variable We use "let someName = value;"

More specifically we can use "let year = 1985;

Now every time we refer to year in the console we get 1985.

So we can declare anything that has a number value by using "let"

We can say let v1 = 5; let v2 = 3; let totalV = v1 + v2; then when we type in totalV it will give us the total

BE AWARE! If there is a change in the number a variable then this will not automatically show up in the total

For example, in our example above, if v2 changed to 6 then the totalV would not change unless we ran let totalV = v1 + v2; again.

It will stay the same as a snapshot of what we previously input.

147. Updating Values

Okay, so we have an example of score. we declare

let score = 2;
So now if we type "score" we get 2.

If we then type

score = score + 2
it updates the score to 4.

However, we can short hand this by using +=. Foe example, we can use

score += 2

We can also use

-=
or
*=
or /=

And then, for even shorter hand we can use

--
or
++
to incrament or decrement by one.

Const & Var

Other than "let" there are 2 other keywords we can use to create variables. They are Const and Var. Const works just like let, except you cannot change the value! You can't change it in the same way that you can change "let" with += or ++ or -=

You can use it store the value of pi, days of the week. General things that are constants. Feet in miles etc.

When we get to arrays and objects and data structures in JS, const will be more useful.

Var is the old variable keyword Before Let and Const, Var was the only way of declaring variables. These days, there isn't really a reason to use it.

BOOLEANS

BOOLEANS ARE very simple, we have two values for booleans - True or False. THat's it. We write the words true or false. we can write

let isActiveGame = true
and then
isActiveGame
would return
true

You can actually just type the variable label and then true of false. so

dinp = false
will return
false

In JavaScript Variables can change type. So if we start with a number e.g.

let numPups = 23;
we can change this to a boolean
numPups = false;
and then back to a number
numPups = 100;
RAre that you might want to do that but you can, if you want to. In other languages if you make a number it remains a number and cannot be changed.
