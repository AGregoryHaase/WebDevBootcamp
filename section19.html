how do we do it?

  • we need to create a let statement variable
    • let input = prompt ("hey, say something!")

    • while(true) {
      I think this is fairly self explanatory?
  • Now we create our response to the input by returning the input as the prompt
    • input = prompt(input)
      this runs the input prompt again but this time with the previous input. At this point we have an infinite loop which is bad
  • Next we need our if statement, which will include our Break Keyword
    • if(input.toLowerCase() === "stop copying me") break;

    • console.log("okay you win"

199. The Lovely For...Of Loop

const subreddits = ['cringe', 'books', 'chickens', 'funny', 'pics', 'soccer']

To print out the above array we could use a 'for' loop.

for (let i = 0; i < subreddits.length; i++){
console.log(`Visit reddit.com/r/${subreddits[i]})
}

But there is now an easier way of doing this, called a For..Or loop

A For..Of loop looks like

for (variable of iterable) {
statement
}

So to rewrite the above array iteration we can go

for(let sub of subreddit) {
console.log(sub)
}
